What is Media Query

Media queries in CSS allow you to apply styles based on device characteristics like screen size or orientation. They are used to make websites adapt to different devices.

Why Media Query is Important

They make websites responsive and improve user experience on all devices. They allow layouts, fonts, and images to adjust automatically for different screens.

What are Keyframes in CSS

Keyframes in CSS are used to create animations by defining how an element changes step-by-step over time.They control the animation's start,middle,and end states(like position,color,size.)
